# Theme structure
```
Themes/Listify Child/                           # → Plugin root
├── images/                                     # → Images folder
│   │── default-badge.png                       # → Default Badge image
├── js/                                         # → JavaScript folder                    
│   └── script.js                               # → JavaScript functions
├── archive-badge-content.php                   # → Content of each post in archive page
├── archive-badge-none.php                      # → No search result in archive page
├── archive-open-badge.php                      # → General content of archive page
├── author.php                                  # → Profile page
├── functions.php                               # → PHP functions for setup the theme
├── screenshot.png                              # → Theme image (displayed in Appearance->Themes)
├── single-badge-content.php                    # → Single badge post content
├── single-open-badge.php                       # → Single badge general content
└── style.css                                   # → Customise style of the theme
